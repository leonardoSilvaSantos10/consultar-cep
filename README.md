📍 Consulta de CEP

Projeto desenvolvido com **HTML, CSS e JavaScript** para consultar endereços a partir de um CEP utilizando a API ViaCEP.

A aplicação valida o CEP informado pelo usuário, realiza a requisição à API e exibe os dados do endereço de forma organizada e responsiva. Durante o desenvolvimento, foram aplicados conceitos de manipulação do DOM, consumo de APIs com `fetch`, programação assíncrona (`async/await`), validação de formulários e estilização com CSS.

🚀 Funcionalidades

* Consulta de CEP em tempo real utilizando a API ViaCEP.
* Validação do CEP (apenas números e 8 dígitos).
* Exibição de logradouro, bairro, cidade e estado.
* Tratamento de mensagens de erro para CEP inválido ou inexistente.
* Interface moderna e responsiva.
* Utilização de ícones com Font Awesome.

🛠️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (ES6+)
* API ViaCEP
* Font Awesome
